## Descripción General del Sistema

Sistema de cajero automático desarrollado en Python que permite a un usuario autenticarse mediante usuario y contraseña y realizar operaciones bancarias básicas como consultar saldo, depositar dinero, extraer fondos y realizar transferencias.

Los datos de la cuenta se almacenan en un archivo de texto para conservar la información entre ejecuciones.
